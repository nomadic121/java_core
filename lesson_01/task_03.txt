3. Попробуйте предсказать результат, а потом проверьте себя:
"" + 1 + 0 = "10" -> конкатенирует как строки
"" - 1 + 0 = -1 -> производит математические операции (0 - 1 + 0)
true + false = 1 -> производит математические операции (1 + 0)
6 / "3" = 2 -> производит математические операции (6 / 3)
"2" * "3" = 6 -> производит математические операции (2 / 3)
4 + 5 + "px" = "9px" -> производит математические операции (4 + 5), затем преобразует в строку и конкатенирует
"$" + 4 + 5 = "$45" -> конкатенирует как строки
"4" - 2 = 2 -> производит математические операции (4 - 2)
"4px" - 2 = nan -> пытается преобразовать "4px" в число и получается nan -> nan -2 = nan
7 / 0 = infinity
"  -9  " + 5 = "  -9  5" -> конкатенирует как строки
"  -9  " - 5 = -14 -> производит математические операции (-9 - 5)
null + 1 = 1 -> null преобразовывает в число 0 и складывает (0 + 1)
undefined + 1 = nan -> undefined не может быть приведено к числу, получаем nan + 1 = nan
" \t \n" - 2 = -2 -> табуляция и перенос строки тримятся, получается пустая строка, которая преобразовывается к нулю и вычитается (0 - 2)
